◆ animationTicker

AnimationTicker RC_Framework.AnimationFrameList.animationTicker =null

A complicated animation class using a timer of class called AnimationTicker and a list of frames of class called FrameList

To use this class its normal to call it as follows:

// first here set up the frame list .... frList FrameList = new FrameList(); frList.add(tex, new Rectangle(0,0,100,100); frList.add(tex, new Rectangle(0,0,200,100); frList.add(tex2, new Rectangle(0,0,100,100);

// now set up the animation ticker int aseq[] = new aseq[3]; AnimationTicker anTicker = new AnimationTicker(aSeq, 0, 3, 9, 0);

// now combine them to what you want AnimationFrameList afl = new AnimationFrameList(frList, anTicker); afl.animationTicker.animationStart();

// finally put them in the sprite or renderable sprite.frameSource = afl;

// finally start the animation using the animation ticker afl.animationTicker.animationStart();
