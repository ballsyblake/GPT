Detailed Description

The Purpose of this class is to provide a canvas (perhaps it sould have been labled RC_canvas) It provides a few routines to work with actual pixels witout needing to load an image use createTex(GraphicsDevice device) to convert it to a texture which is what you need to actually render the surface Surfaces are always rectanglar
